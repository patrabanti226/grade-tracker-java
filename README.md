🎓 Student Grade Tracker (Java Swing)
📖 Project Overview

The Student Grade Tracker is a simple Java GUI application that allows users to enter student names and marks and generate a grade report.
The application is built using Java Swing and demonstrates basic concepts of GUI development and event handling in Java.

This project is useful for learning how to build desktop applications using Java.

✨ Features

Add student name and marks

Store multiple student records

Generate grade reports

Simple and user-friendly interface

GUI built using Java Swing

🛠 Technologies Used

Java

Java Swing (GUI)

Object-Oriented Programming (OOP)

Event Handling
